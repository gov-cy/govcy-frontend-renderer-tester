Τι κάνει αυτή η υπηρεσία

Μπορείτε να δώσετε την συγκατάθεση σας για την έκδοση ή επανέκδοση εγγράφων ανηλίκων κάτω των 12 χρόνων.

Για ποιους είναι η υπηρεσία

Μπορείτε να χρησιμοποιήσετε την υπηρεσία αν:

  • είστε πάνω από 18 χρόνων
  • είστε Κύπριος ή Κύπρια

Τι θα χρειαστείτε για να χρησιμοποιήσετε την υπηρεσία

Θα χρειαστείτε λογαριασμό στο CY Login (πρώην Αριάδνη). Χρειάζεται να είναι ταυτοποιημένος.
